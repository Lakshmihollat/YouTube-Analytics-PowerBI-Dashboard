# YouTube Analytics Dashboard using Power BI

This project presents an interactive Power BI dashboard designed to analyze
YouTube channel and video performance using key analytics metrics.
The dashboard transforms raw YouTube data into meaningful insights through
dynamic visualizations and KPI-based analysis.

## Project Overview
The objective of this project is to provide a clear and intuitive way to
evaluate YouTube content performance without using complex predictive models.
The system focuses on accurate computation of engagement and performance
metrics along with interactive data exploration.

## Features
- KPI-based analysis including:
  - Total Views
  - Average Engagement Percentage
  - Views Per Day
  - Like Ratio Percentage
  - Comment Ratio Percentage
  - Viral Score
- Interactive filters and slicers for channel and video comparison
- Visual analysis using KPI cards, donut charts, area charts, and scatter plots
- Clean and minimal dashboard design for better readability

## Tools & Technologies Used
- Power BI Desktop
- Power Query (Data Cleaning and Transformation)
- DAX (Data Analysis Expressions)
- CSV/Excel Dataset

## Dashboard Visuals
- KPI Cards for performance summary
- Donut Charts for video definition (HD/SD) and caption availability
- Area Charts to analyze upload trends over time
- Scatter Plots to compare video duration vs engagement

## How to Use
1. Download the `.pbix` file from this repository
2. Open it using **Power BI Desktop**
3. Refresh the data if required
4. Use slicers and filters to explore insights

## Project Outcome
The dashboard successfully converts raw YouTube analytics data into an
interactive and user-friendly analytical system. It enables efficient
evaluation of content performance and supports data-driven insights for
creators and analysts.
